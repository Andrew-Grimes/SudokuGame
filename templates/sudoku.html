Directions

Start a Game

Select a difficulty level (Easy, Medium, or Hard).

The game generates a board with some numbers already filled in.

Placing Numbers

Click on an empty square and enter a number (1-9).

If the number is correct, it remains bolded.

If the number is incorrect, it is removed, and a strike is added.

Highlighting

Click on a pre-filled number, and all visible instances of that number will be highlighted with a mint gray background and bold black border.

This helps track numbers easily across the board.

Winning the Game

The puzzle is complete when all cells are correctly filled.

Click Check Solution to verify your answers.

If all numbers are correct, you win the game!

Losing the Game

If you exceed the allowed number of strikes (if implemented), you may lose the game.
